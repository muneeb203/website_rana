# Product Overview

ConvoSol is a custom software development company landing page showcasing services and portfolio.

## Purpose
Marketing website to attract clients for custom software solutions including web apps, mobile apps, cloud infrastructure, and AI solutions.

## Key Features
- Service showcase (Web, Mobile, Cloud/DevOps, AI)
- 4-phase process timeline (Discover, Design, Develop, Deliver)
- Portfolio case studies with tech stack tags
- Company stats and values
- Contact form for lead generation

## Target Audience
Startups and enterprises seeking custom software development services.
